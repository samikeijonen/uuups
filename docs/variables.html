Baseline grid lines height is so called 8 point grid. Every spacing metric should be based on this. Inspired by inuitcss.

Related

: https://github.com/inuitcss/inuitcss

$global-factor-1

scss 
$global-factor-1: 1 !default;

How many grid lines should our spacing unit variants span? Each value should be an unitless integer.

$global-spacing-unit-1

scss 
$global-spacing-unit-1: $global-baseline * $global-factor-1 !default;

Spacing values are determined based on your project’s global baseline grid. It is not recommended that you modify these following variables (it can break your vertical rhythm), but if you need to, you can.

@mixin font-size()

Generates a rem font-size and a baseline-compatible unitless line-height from font-size value.

You can also modify the line-height by increments, while staying in the baseline grid, by setting the $modifier parameter.

It takes a positive or negative number, and it will add or remove “lines” to the generated line-height. This is the recomended way to do it, unless you really need an absolute value.

It is recommended to only add or subtract full integers or half steps.

Parameters & Return

$font-size: (number)

$line-height: auto (number)

Line height

$modifier: 0 (number)

Line modifier

@error

#{$font-size} needs to be a number.

@error

D’oh! #{$line-height} is not a valid value for $line-height.

Examples

scss Example 1: 
@include font-size(1.25rem);
scss Example 2, force line-height: 
@include font-size(1rem, 1.5);
scss Example 3, add 2 lines: 
@include font-size(2rem, $modifier: +2);
scss Example 4, subtract -0.5 lines: 
@include font-size(0.875rem, $modifier: -0.5);

Requires

$global-font-size

@mixin center()

Center any element.

Examples

scss Basic Usage Sass 
.center {
  @include center;
  }
css Basic Usage CSS Output 
.center {
  margin-left: auto;
  margin-right: auto;
}
